# Simple Screen Recorder

シンプルな画面録画Chrome拡張機能

## 機能

- 画面全体、ウィンドウ、またはタブの録画
- 音声録音のON/OFF
- WebM形式での自動保存

## インストール

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をONにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使い方

1. 拡張機能アイコンをクリック
2. 「録画開始」ボタンをクリック
3. 録画したい画面を選択
4. 「録画停止」ボタンで終了

録画ファイルは自動的にダウンロードフォルダに保存されます。

## ファイル構成

- `manifest.json` - 拡張機能の設定
- `popup.html` - ポップアップUI
- `popup.js` - 録画機能の実装
- `styles.css` - スタイル定義
- `icon.png` - 拡張機能アイコン
